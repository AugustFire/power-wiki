Atlas 是我们视觉语言的唯一事实来源。它确保了跨平台的一致性，使工程师和设计师能够专注于逻辑实现，而无需重新创建 UI 模式。

ℹ️
文档说明： 此版本引入了 OKLCH 颜色渐变和精细化的间距令牌。有关迁移步骤，请参阅 v2 到 v3 迁移指南。

设计原则

这些原则指导我们在构建或扩展系统时做出的每一项决定：

  • 清晰性 (Clarity)： 界面应该是直观且功能明确的。
  • 速度 (Speed)： 优先交付可用的工具，而非追求绝对的完美。
  • 无障碍 (Accessibility)： WCAG 2.2 AA 是我们的基准要求。

颜色令牌 (Color Tokens)

色板锚定在 Atlassian 标准的蓝色和中性色中，以确保熟悉、专业的环境感。

品牌蓝 (Brand Blue)
#0052CC
柔和蓝 (Soft Blue)
#DEEBFF
成功绿 (Success Green)
#36B37E
警告黄 (Warning Amber)
#FFAB00

字体排版

我们使用 Plus Jakarta Sans 处理 UI 和长篇文本，使用 JetBrains Mono 进行技术参考。

⚠️
弃用警告： v2.x 中使用的 "Inter" 字体已正式弃用。所有新项目必须加载 Plus Jakarta 字体。

令牌使用

令牌以 CSS 变量的形式导出，是样式化组件的首选方式。

:root {
  --color-primary: #0052CC;
  --color-text: #172B4D;
  --space-8: 8px;
}

.button {
  background: var(--color-primary);
  padding: var(--space-8);
}
